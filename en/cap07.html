Chapter 7 – Diameter (4G) in Depth

If SS7 is the “nervous system” of legacy networks, then Diameter is the signaling brain of 4G.

In this chapter, you will understand how 4G works internally, why it is more advanced than SS7 — and why it can still fail in critical ways.

7.1 What is Diameter?

Diameter is the signaling protocol used in the 4G (LTE) core network. It controls:

  • User authentication
  • Mobility (network registration)
  • Usage policies (internet, QoS)
  • Billing (charging)

Difference from SS7:

  • SS7 → based on implicit trust
  • Diameter → based on sessions and authentication

👉 However, it still depends on trust between operators.

7.2 LTE core architecture

Element Function
MME Manages mobility and authentication
HSS Subscriber database
SGW Routes traffic within the network
PGW Connects to the internet
PCRF Defines policies and charging rules

👉 Diameter connects all these elements.

7.6 The main risk of Diameter

Although more modern than SS7, Diameter still has issues:

  • High complexity
  • System interdependencies
  • Automatic retries
  • Trust between roaming partners

👉 This creates a perfect scenario for large-scale failures.

7.7 Diameter Storm (in depth)

⚠️ Definition

A Diameter Storm occurs when a massive volume of messages overwhelms the network.

🧠 How does it start?

  • Failure in a network element (e.g., HSS)
  • Timeout responses
  • MME retrying requests

🔗 Cascade effect

StepDescription
1Initial HSS failure
2MME sends multiple retries
3Other MMEs do the same
4Traffic grows exponentially
5Network collapses

Result:

  • Users cannot connect
  • Sessions drop
  • Network instability

7.12 Business impact

  • Network outages (millions of users)
  • Revenue loss
  • Critical service disruption
  • Reputation damage

👉 Unlike SS7, the problem here is scale.

7.13 Conclusion

Diameter brought major improvements, but also new risks:

  • More advanced architecture
  • Extremely complex system
  • Susceptible to cascading failures

In 4G, the biggest enemy is not only the attacker — but the network’s own domino effect.
